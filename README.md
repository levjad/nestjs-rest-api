## Description

This is a NestJS rest api project.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Swagger

This project uses swagger ui. To open the UI install all dependencies and start the app. The UI is accessible via http://localhost:3000/api.
